Pharma Commercial Analytics System

End-to-end analytics pipeline built on synthetic Indian pharma 
distribution data using MySQL, Power BI, and Excel.

 Tools Used
- MySQL — database and query layer
- Power BI — 5-page executive dashboard
- Excel — management reporting layer

 Database Schema
7-table star schema covering products, geographies, channels, 
sales reps, dates, sales transactions, and inventory.
- 2,500 sales records | 1,500 inventory records | 28 Indian states

 SQL Queries
1. Revenue by product category and region
2. Sales rep target achievement
3. Top 5 products by revenue
4. Channel mix analysis
5. Monthly revenue trend
6. Stockout risk identification

Key Business Insights
- Ferikind is highest revenue product at 4.36% portfolio contribution
- Hospital channel drives highest revenue across all regions
- West region shows lowest revenue — demand gap, not supply issue
- 102 inventory instances at stockout risk across distribution network
- North region leads target achievement at 93.6%
